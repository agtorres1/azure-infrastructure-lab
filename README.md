# azure-infrastructure-lab
Azure infrastructure deployment and administration project based on AZ-104 concepts.

## Overview
This project demonstrates the deployment and administration of a cloud infrastructure in Microsoft Azure based on AZ-104 concepts.

## Objectives
- Deploy and manage Azure resources
- Configure virtual networking
- Implement security controls
- Monitor infrastructure performance
- Apply governance best practices

## Services Used
- Azure Virtual Network
- Network Security Groups
- Azure Virtual Machines
- Azure Monitor
- Azure Storage
- Azure RBAC

## Skills Demonstrated
- Cloud infrastructure deployment
- Network segmentation
- Access control implementation
- Monitoring and alerting
- Linux administration in Azure
